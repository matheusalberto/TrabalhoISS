Ajuda - Pedido

Ajuda para o pedido.

  1. Escolha entre:
    • Cadastrar um cliente
    • Escolher um cliente cadastrado digitando as iniciais do seu nome, selecionando o cliente e pressionando o botão ou a tecla ENTER do teclado.
  2. Digite as iniciais do item no campo para ser mostrado na tabela ou pressione a tecla ENTER para listar todos os itens em estoque?
    • Para adicionar itens a cesta selecione o item na tabela de Produtos e depois pressione o botão
    • Para remover itens da cesta, selecione o item a ser removido na tabela da cesta e pressione o botão
  3. Pressione o botão para finalizar a compra ou o botão para cancelar o pedido
